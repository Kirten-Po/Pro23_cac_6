|   |   |
|---|---|
|Ограничение времени|1 секунда|
|Ограничение памяти|64 Мб|
|Ввод|стандартный ввод или cities.txt|
|Вывод|стандартный вывод или eclipse.json|

Частичное солнечное затмение происходит в узкой полосе на поверхности Земли последовательно в одних пунктах за другими. В файле **cities.txt** записаны строки с городами и временем начала видимости затмения в формате:  
_город час:минуты_

В файл **eclipse.json** запишите список городов в последовательности, в которой в них начинается солнечное затмение.

## Пример

|Ввод|Вывод|
|---|---|
|Nuuk 08:53<br>Halifax 07:17<br>St.John's 07:52|["Halifax", "St.John's", "Nuuk"]|

## Примечания

В примере ввода и вывода показано содержимое файлов.

import json


with (open('cities.txt') as in_file,
      open('eclipse.json', 'w') as out_file):
    data = in_file.readlines()
    data = [x.rstrip().split() for x in data]
    result = []
    for x in data:
        digit = x[1].split(':')
        sum_num = int(digit[0]) * 60 + int(digit[1])
        result.append([x[0], sum_num])
    result.sort(key=lambda x: x[1])
    lst = [x[0] for x in result]
    json.dump(lst, out_file)
