Мы уже встречались с очень удобным инструментом распаковки коллекций — звёздочкой, когда нужно было вывести все значения коллекции подряд. Например, вот так:

```python
arr = [1, 2, 5, 7, 8, 0]
print(*arr)  # 1 2 5 7 8 0
```

А ещё пользовались множественным присваиванием, когда с помощью одного знака равенства присваивали значения сразу нескольким переменным:

```python
n, m, k = 1, 2, 3
```

В этом примере переменные слева запаковываются в кортеж и этому кортежу присваивается значение кортежа, находящегося справа от знака равенства. А затем оба кортежа распаковываются, и каждая переменная получает своё значение.

Но справа можно поставить не только кортеж, а любую другую коллекцию:

```python
n, m, k = 'abc'
n, m, k = {3, 4, 5}
n, m, k = ['7', '8', 'F']
```

Главное, чтобы количество переменных слева было такое же, как и количество значений справа.

Но что, если число значений справа заранее неизвестно? Тогда можно использовать переменную со звёздочкой (звёздочка ставится слева от имени переменной). Например, мы знаем, что в коллекции будет не менее одного значения, но может быть и больше. Тогда значение первого элемента присвоим переменной n, а все остальные запакуем в список и запишем в переменную rest:

```python
n, *rest = [1, 2, 3, 4]
print(n, rest)  # 1 [2, 3, 4]
```

или так:

```python
n, *rest = 'abc'
print(n, rest)  # a ['b', 'c']
```

Если в коллекции окажется всего одно значение, то оно запишется в переменную n, а список останется пустым. Но вот если коллекция будет совсем пустой, то программа выдаст ошибку, так как распаковывать будет нечего!

Звёздочка может стоять и в начале, и в середине:

```python
*rest, n = [1, 2, 3, 4]
print(n, rest)  # 4 [1, 2, 3]
```

или так:

```python
n, *rest, m = 1, 2, 3, 4, 5
print(n, rest, m)  # 1 [2, 3, 4] 5
```

Важно, чтобы интерпретатору было понятно, какие значения в какие переменные записывать. Подумайте, можно ли однозначно понять, в каких переменных окажутся значения при такой распаковке:

**Неверно**

```python
n, *rest, *m = 1, 2, 3, 4, 5
```

**Верно**

```python
n, *rest, m, k = 1, 2, 3, 4, 5
```

Также можно распаковывать вложенные списки:

```python
a, (b, c, d), e, f = [1, [2, 3, 4], 5, 6]
print(a, b, c, d, e, f)  # 1 2 3 4 5 6
```

Важно следить, чтобы переменные слева и значения справа были _согласованы_. Лучший способ понять, как происходит запаковка и распаковка значений, — это поэкспериментировать с ними.

Вернёмся теперь к самому началу. Вспомните, как мы использовали звёздочку для распаковки аргумента в функции print():

```python
arr = [1, 2, 5, 7, 8, 0]
print(*arr)  # 1 2 5 7 8 0
```

В этом случае звёздочка не _запаковывает_, а распаковывает элементы коллекции, затем выводит их все через разделитель, который по умолчанию равен пробелу, но это можно изменить:

```python
arr = [1, 2, 5, 7, 8, 0]
print(*arr, sep=', ')  # 1, 2, 5, 7, 8, 0
```
