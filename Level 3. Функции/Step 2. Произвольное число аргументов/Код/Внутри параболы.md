Напишите функцию, принимающую три числа и произвольное число кортежей из двух чисел — координат точек `x` и `y`. Первые три числа означают коэффициенты параболы `a`, `b` и `c`:

y=ax2+bx+cy=ax2+bx+c

Функция _parabola()_ возвращает список кортежей точек, которые находятся внутри параболы, то есть выше неё, если ветви параболы направлены вверх, или ниже неё, если ветви направлены вниз. Кортежи в списке идут в исходном порядке.

## Пример

|Ввод|Вывод|
|---|---|
|points = [(0, 0), (1.2, 3), (-1, -2)]<br>print(parabola(1, -2, -3, *points))|[(0, 0), (1.2, 3)]|

def parabola(a, b, c, *xy):
    lst = []
    for i in xy:
        y = a * i[0] ** 2 + b * i[0] + c
        if a > 0:
            if i[1] >= y:
                lst.append(i)
        elif a < 0:
            if i[1] <= y:
                lst.append(i)
    return lst
