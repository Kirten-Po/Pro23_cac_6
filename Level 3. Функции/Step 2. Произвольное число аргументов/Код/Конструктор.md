Функция _builder()_ принимает число и произвольное количество аргументов-строк. Возвращает слово, которое получится, если из каждого аргумента-строки взять _n_-й символ.

Напишите эту функцию.

## Пример

|Ввод|Вывод|
|---|---|
|print(builder(2, 'Yellow', 'flame', 'flickers', 'Shadows', 'dance', 'upon', 'the', 'wall'))|laianoel|

def builder(n, *args):
    lst = [word[n] for word in args]
    return ''.join(lst)
