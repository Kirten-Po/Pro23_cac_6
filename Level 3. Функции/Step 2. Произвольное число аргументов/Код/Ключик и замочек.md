Ключик и замочек — это числа.

Ключик подходит к замочку, если его можно получить из замочка, выбросив некоторые цифры.

Напишите функцию _key_and_lock()_, принимающую число-ключик и произвольное число потенциальных замочков. Функция возвращает список чисел-замочков для этого ключика в исходном порядке.

## Пример

|Ввод|Вывод|
|---|---|
|print(key_and_lock(12345, *(2100231400566, 1203056, 331203045)))|[2100231400566, 331203045]|

def key_and_lock(n, *args):
    f = str(n)
    lst = []
    r = []
    for g in args:
        g1 = str(g)
        j = 0
        for i in range(len(g1)):
            if g1[i] == f[j]:
                lst.append(g1[i])
                if j != len(f) - 1:
                    j += 1 
        r1 = ''.join(lst)    
        r.append(r1)
        lst.clear()
    lst = [args[j] for j in range(len(r)) if f in r[j]]
    return lst
