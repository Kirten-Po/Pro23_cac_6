Кактус ощетинился колючками — выставил их навстречу опасностям.

Напишите функцию _cactus()_, которая из произвольного числа переданных аргументов выбирает числа-кактусы:  
первая и последняя цифры таких чисел не должны быть делителями числа, которое останется, если первую и последнюю цифру удалить.

Функция должна вернуть список чисел-кактусов в исходном порядке.

## Пример

|Ввод|Вывод|
|---|---|
|print(*cactus(12345, 23, 2773, 51336))|2773 51336|

def cactus(*n):
    lst = []
    for i in n:
        if len(str(i)) > 1:
            n1, *n2, n3 = str(i)
            n2 = ''.join(n2)
            if len(n2) != 0 and n3 != '0':
                if int(n2) % int(n1) != 0 and int(n2) % int(n3) != 0:
                    lst.append(i)
    return lst
