Функция может принимать произвольное число не только позиционных, но и именованных аргументов. Записывается это так:

```python
def func(*args, **kwargs):
    return args, kwargs
```

Если все позиционные аргументы запаковываются в кортеж, то для именованных создаётся словарь: имя аргумента — это ключ, значение — значение по этому ключу.

```python
arr = ['abc','Hello', 'hi']
print(*func(*arr, order=len, name='func', nums=None), sep='\n')
# (123, 96, 47, 19, 29, 24, 96)
# {'order': <built-in function len>, 'name': 'func', 'nums': None}
```

В задаче нам не всегда известно, какие именно именованные аргументы будут переданы. Это нужно проверять в программе. Например, так, как вот в этой задаче.

## Пример задачи с произвольным числом именованных аргументов

Белка Сирле набрала орехов и шишек для своих бельчат. Но в гости может заглянуть Нильс, а то и другие друзья семьи. Поэтому могут быть варианты, что и на сколько частей нужно разделить.

Напишите функцию **splitting()**, которая принимает список шишек, произвольное число орехов и произвольное число именованных аргументов, среди которых могут быть такие:

squirrels — количество бельчат, по умолчанию 3;

guests — количество гостей;

prefer — любят ли гости шишки, или только орехи; если значение cones, то любят и шишки тоже, если значение nuts, то любят только орехи; если аргумент не передан, то гости не голодны, ничего есть не будут.

Функция должна вернуть два числа: по сколько штук каждому, кто ест орехи, достанется орехов, и по сколько штук шишек достанется тем, кто ест шишки.

Сумму всех орехов надо нацело разделить на _количество бельчат + количество гостей + белка Сирле_, если гости едят орехи, иначе только на бельчат и белку.

Сумму всех шишек нужно нацело разделить на бельчат, белку, на гостей в том случае, если гости их едят.

_Примеры для проверки решения_

|   |   |
|---|---|
|```python<br>cone = [12, 13, 5, 7]<br>nut = [3, 5, 17, 4, 8]<br>print(splitting(cone, *nut, guests=2))<br>```|```python<br>(9, 9)<br>```|
|```python<br>cone = [12, 13, 5, 7]<br>nut = [3, 5, 17, 4, 8]<br>print(splitting(cone, *nut, prefer='nuts', guests=2))<br>```|```python<br>(6, 9)<br>```|
|```python<br>cone = [12, 13, 5, 7]<br>nut = [3, 5, 17, 4, 8]<br>print(splitting(cone, *nut, prefer='nuts', guests=2, squirrels=4))<br>```|```python<br>(5, 7)<br>```|

Решение

```python
def splitting(cones, *nuts, squirrels=3, **kwargs):
    if 'prefer' not in kwargs:
        return sum(nuts) // (squirrels + 1), sum(cones) // (squirrels + 1)
    if kwargs['prefer'] == 'cones':
        return sum(nuts) // (squirrels + 1), sum(cones) // (kwargs['guests'] + squirrels + 1)
    return sum(nuts) // (kwargs['guests'] + squirrels + 1), sum(cones) // (squirrels + 1)
```
