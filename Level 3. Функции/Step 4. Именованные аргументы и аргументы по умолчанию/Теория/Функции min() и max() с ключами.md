Функции min() и max(), кроме коллекции для поиска наименьшего и наибольшего значений соответственно, могут принимать именованный аргумент key, который указывает, по какому ключу произвести сортировку этой коллекции, чтобы затем взять нужный элемент.

Если передать в функцию список строк без ключа, то сравниваться они будут лексикографически. Если же нам нужно найти наименьшую строку при сравнении по длине, то аргументу key передадим встроенную функцию len:

```python
lst = ['one', 'two', 'three', 'four', 'five', 'six', 'seven']
print(sorted(lst, key=len))
# ['one', 'two', 'six', 'four', 'five', 'three', 'seven']
print(min(lst, key=len))  # one
print(max(lst, key=len))  # three
```

Важно, что при таком применении обе функции вернут строку, а не её длину. К тому же если строк с одинаковой длиной окажется несколько, то будет возвращена первая подходящая.

Строк с наименьшей длиной оказалось две — one и two, возвращена первая.

Строк с наибольшей одинаковой длиной тоже две, возвращена первая — three.

Как и в sorted(), функцию для поиска минимума или максимума можно написать самостоятельно. Воспользуемся поиском по количеству гласных, который мы уже написали. Тогда результат будет такой:

```python
def count_vowels(s):
    return len([x for x in s if x.lower() in 'aeyuio'])


lst = ['one', 'two', 'three', 'four', 'five', 'six', 'seven']
print(sorted(lst, key=count_vowels))
# ['two', 'six', 'one', 'three', 'four', 'five', 'seven']
print(min(lst, key=count_vowels))  # two
print(max(lst, key=count_vowels))  # one
```

Среди строк с наибольшим количеством гласных в списке первым шло one, его мы и получили при поиске максимума.
