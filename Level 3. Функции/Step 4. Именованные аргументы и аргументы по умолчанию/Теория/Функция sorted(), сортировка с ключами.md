Для сортировки списка мы использовали метод sort(), однако, чтобы отсортировать другие коллекции, приходилось сначала превращать их в список. Есть способ лучше — воспользоваться функцией sorted().

Функция принимает любую коллекцию и именованный аргумент key, превращает коллекцию в список и по умолчанию сортирует его по возрастанию. Функция не меняет исходную коллекцию, как это делает метод sort(), она возвращает отсортированный список, какую бы коллекцию мы ему ни передали.

**Важно!**

В коллекции должны быть однотипные элементы, чтобы их можно было сравнивать между собой.

Сортируем список, множество, строку, словарь (конечно же, только его ключи):

```python
lst = ['one', 'two', 'three', 'four', 'five', 'six']
print(sorted(lst))
# ['five', 'four', 'one', 'six', 'three', 'two']

s = 'eleven'
print(sorted(s))  # ['e', 'e', 'e', 'l', 'n', 'v']

st = {'lizard', 'snake', 'parrot', 'horse'}
print(sorted(st))  # ['horse', 'lizard', 'parrot', 'snake']

dic = {'milk': 3.6, 'cream': 6.0, 'butter': 82.5}
print(sorted(dic))  # ['butter', 'cream', 'milk']
```

Кроме того, можно менять ключ сортировки. Так, если нужно строку отсортировать не по алфавиту, а, например, по длине, то ключу нужно передать имя функции для сортировки (**не вызывать!** скобочки не ставить!):

```python
lst = ['one', 'two', 'three', 'four', 'five', 'six']
print(sorted(lst, key=len))
# ['one', 'two', 'six', 'four', 'five', 'three']
```

Важно понимать, что если элементы имеют одинаковые значения при сортировке по ключу (например, одинаковую длину), то их порядок не меняется, остаётся таким же, как был в исходном списке.

Кроме key, функция sorted() имеет ещё один именованный аргумент — reverse со значением по умолчанию False, который означает, что сортировать нужно в прямом порядке. Если передать ему значение True, то порядок сортировки сменится на обратный:

```python
lst = ['one', 'two', 'three', 'four', 'five', 'six']
print(sorted(lst, key=len, reverse=True))
# ['three', 'four', 'five', 'one', 'two', 'six']
```

## Сортировка с собственным ключом

Что делать, если для сортировки, которая требуется в задаче, нет встроенной функции? Например, слова нужно сортировать по количеству гласных букв? Ну что ж, напишем свою функцию, возвращающую количество таких букв в строке, и передадим её имя именованному аргументу key:

```python
def count_vowels(s):
    return len([x for x in s if x.lower() in 'aeyuio'])


lst = ['one', 'two', 'three', 'four', 'five', 'six', 'seven']
print(sorted(lst, key=count_vowels))
# ['two', 'six', 'one', 'three', 'four', 'five', 'seven']
```

А если нужно отсортировать не в порядке увеличения количества гласных, а в порядке уменьшения, то в ключевой функции перед возвращаемым значением поставим минус.

```python
def count_vowels(s):
    return -len([x for x in s if x.lower() in 'aeyuio'])
```

Почему это работает? Функция возвращает число (количество гласных), а порядок отрицательных чисел противоположен порядку положительных с таким же абсолютным значением.

Для других типов данных (не чисел) такой приём, конечно, не работает.
