Добавьте новую ступеньку к лесенке из строк.

Напишите функцию _stairs()_, которая принимает аргумент-строку и именованный аргумент-список `steps`, по умолчанию пустой.

Функция добавляет строку в начало списка, если первый его элемент лексикографически больше строки, или в конец списка, если последний элемент меньше строки, или не добавляет вообще, если ни одно условие не выполнено. Если список пуст, то строка-аргумент в него добавляется. Затем функция возвращает изменённый список.

### Пример 1

|Ввод|Вывод|
|---|---|
|print(stairs('one', steps=['two']))|['one', 'two']|

### Пример 2

|Ввод|Вывод|
|---|---|
|print(stairs('six', steps=['four', 'five']))|['four', 'five', 'six']|

### Пример 3

|Ввод|Вывод|
|---|---|
|print(stairs('three'))|['three']|


def stairs(a, steps=[]):
    if not steps:
        steps.append(a)
    elif a < steps[0]:
        steps.insert(0, a)
    elif a > steps[-1]:
        steps.append(a)
    return steps
