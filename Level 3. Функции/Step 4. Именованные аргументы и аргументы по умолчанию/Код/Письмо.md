Напишите функцию _letter()_ для вставки значений в шаблон письма. Функция принимает аргументы:  
текст сообщения;  
`name` — именованный аргумент со значением по умолчанию _Ivan_;  
`surname` — именованный аргумент со значением по умолчанию _Ivanov_;  
`address` — именованный аргумент со значением по умолчанию _Moscow_;  
`from_whom` — именованный аргумент со значением по умолчанию _John_.

Функция собирает из аргументов строку по шаблону и возвращает её:

```
Where to: <address>
Dear <name> <surname>!
<текст сообщения>
From <from> with best regards!
```

### Пример 1

|Ввод|Вывод|
|---|---|
|print(letter('I invite you to London!'))|Where to: Moscow<br>Dear Ivan Ivanov!<br>I invite you to London!<br>From John with best regards!|

### Пример 2

|Ввод|Вывод|
|---|---|
|print(letter('I invite you to London!', address='Tver', name='Mike'))|Where to: Tver<br>Dear Mike Ivanov!<br>I invite you to London!<br>From John with best regards!|

def letter(args, name='Ivan', surname='Ivanov', address='Moscow', from_whom='John'):
    return f'Where to: {address}\nDear {name} {surname}!\n{args}\nFrom {from_whom} with best regards!'
