Напишите функцию _max_slice()_, которая принимает произвольное число строк и возвращает наибольшую из них, если сравнивать по срезу строки с нечётными индексами.

## Пример

|Ввод|Вывод|
|---|---|
|print(max_slice('first', 'second', 'third', 'forth'))|forth|

def line(*a):
    lst = [x[1::2] for x in a]
    return lst


def max_slice(*a):
    return max(a, key=line)
