Напишите функцию _binary_units()_, которая принимает произвольное количество целых чисел и сортирует их по количеству единиц в их двоичном представлении. Если количество единиц одинаковое, порядок не должен измениться.

Функция возвращает список отсортированных чисел.

## Пример

|Ввод|Вывод|
|---|---|
|print(binary_units(10, 9, 12, 16, 7, 5, 3, 1024))|[16, 1024, 10, 9, 12, 5, 3|

def binary_units(*a):
    return sorted(a, key=lambda x: bin(x).count('1'))
