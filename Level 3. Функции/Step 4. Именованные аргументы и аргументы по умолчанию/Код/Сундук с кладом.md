Пираты нашли сундук с кладом, и чтобы они не передрались, пока будут его делить, главарь решил установить правила, по которым каждому достанется одинаковая часть каждого вида добычи, а всё, что не разделилось нацело, отойдёт главарю.

Напишите функцию, которая разделит все виды добычи “по справедливости”.

Функция _treasure()_ принимает количество пиратов (без главаря), произвольное число чисел — количество добычи каждого вида — и произвольное число именованных аргументов, среди которых могут быть такие:  
`exclude` — исключения — если аргумент передан и добычи какого-то вида меньше значения аргумента, она не делится и вся переходит главарю;  
`special` — особый случай — если аргумент передан и добыча какого-то вида кратна значению аргумента, то перед дележом это значение аргумента переходит главарю, а остальное уже делится по обычному правилу. Именованные аргументы проверяются именно в том порядке, в котором описаны в задаче; второй проверяется, если первого нет или не применяется.

Функция возвращает два списка: количество добычи каждого вида, которое достанется рядовому пирату, и количество, которое достанется главарю.

### Пример 1

|Ввод|Вывод|
|---|---|
|print(*treasure(4, 12, 7, 3, 8))|[3, 1, 0, 2] [0, 3, 3, 0]|

### Пример 2

|Ввод|Вывод|
|---|---|
|print(*treasure(5, 12, 7, 4, 9, exclude=5))|[2, 1, 0, 1] [2, 2, 4, 4]|

### Пример 3

|Ввод|Вывод|
|---|---|
|print(*treasure(3, 12, 7, 4, 9, special=3))|[3, 2, 1, 2] [3, 1, 1, 3]|

def treasure(num, *args, exclude=None, special=None):
    pirates = []
    boss = []
    for item in args:
        if exclude is not None and item < exclude:
            pirates.append(0)
            boss.append(item)
        elif special is not None and item % special == 0:
            remaining_gold = item - special
            pirates.append(remaining_gold // num)
            boss_share = special + (remaining_gold % num)
            boss.append(boss_share)
        else:
            pirates.append(item // num)
            boss.append(item % num)   
    return [pirates, boss]
