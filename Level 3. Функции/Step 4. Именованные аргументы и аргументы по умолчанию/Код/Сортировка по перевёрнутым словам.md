Напишите функцию, которая сортирует набор строк, если их рассматривать задом наперёд и не обращать внимания на регистр.

Функция _reversed_words()_ принимает произвольное число строк и возвращает список из них, но уже отсортированный.

## Пример

|Ввод|Вывод|
|---|---|
|print(reversed_words('Sweet', 'Spring', 'always', 'wins', 'her', 'way', 'Spring', 'brings', 'happiness'))|['Spring', 'Spring', 'her|


def reversed_words(*a):
    return sorted(a, key=lambda x: x.lower()[::-1])
