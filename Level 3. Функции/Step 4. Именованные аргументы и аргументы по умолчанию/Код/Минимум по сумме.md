Напишите функцию _min_by_sum()_, которая принимает произвольное число строк и возвращает наименьшую из них при сравнении по сумме числовых кодов составляющих её символов (их можно получить с помощью функции _ord()_).

## Пример

|Ввод|Вывод|
|---|---|
|print(min_by_sum('first', 'second', 'third', 'forth'))|third|

def min_by_sum(*a):
    return min(a, key=lambda x: sum(ord(b) for b in x))
