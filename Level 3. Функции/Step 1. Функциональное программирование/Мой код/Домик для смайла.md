Напишите функцию _smile_home()_, выводящую домик для смайлика, вот такой:

![home](https://yastatic.net/s3/lyceum/files/e370bae9-d786-4d23-ae91-4da337316943/upload.png)

Смайлик можно вывести в коде _Python_, если написать так:

```
\N{slightly smiling face}
```

## Примечания

В тестирующую систему сдайте **только код функции** без вызова.

def smile_home():
    a = ' '
    for i in range(3, -1, -1):
        print(a * i + '/' + a * (3 - i) * 2 + "\\")
    print('-' * 8)
    print('|' + ' ' * 2 + '\N{slightly smiling face}' + ' ' * 2 + '|')
    print('-' * 8)
