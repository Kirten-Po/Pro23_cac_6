Напишите функцию _three_divisors()_, которая возвращает три наибольших делителя числа в порядке убывания, считая делителями единицу и само число. Если делителей меньше, то вместо недостающих делителей нужно вернуть None.

### Пример 1

|Ввод|Вывод|
|---|---|
|n = 24<br>print(*three_divisors(n))|24 12 8|

### Пример 2

|Ввод|Вывод|
|---|---|
|n = 17<br>print(*three_divisors(n))|17 None 1|

## Примечания

В тестирующую систему сдайте **только код функции** без вызова

def three_divisors(n):
    lst = [i for i in range(n, 0, -1) if n % i == 0][:3]
    lst1 = []
    for i in range(len(lst)):
        if len(lst) <= 2:
            if lst[i] == min(lst):
                lst1.append(None,)
                lst1.append(lst[i])
                break
            else:
                lst1.append(lst[i])
        else:
            lst1 = lst
            break
    return lst1
