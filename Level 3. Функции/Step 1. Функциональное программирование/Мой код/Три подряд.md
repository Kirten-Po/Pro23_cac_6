Напишите функцию _ten_sum()_, которая находит в переданном в аргументе списке три идущих подряд числа, сумма которых равна 10. Функция возвращает индекс первого из этих чисел или -1, если такой тройки чисел не нашлось.

### Пример 1

|Ввод|Вывод|
|---|---|
|arr = [2, 4, 9, 0, 1, 3, 6]<br>print(ten_sum(arr))|2|

### Пример 2

|Ввод|Вывод|
|---|---|
|arr = [1, 7, 5, 4]<br>print(ten_sum(arr))|-1|

## Примечания

В тестирующую систему сдайте **только код функции** без вызова

def ten_sum(arr):
    for i in range(len(arr)):
        if sum(arr[0:3]) != 10:
            arr = arr[1:]
        else:
            return i
    return -1
