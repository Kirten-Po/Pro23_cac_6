Напишите функцию _count_vowels()_, которая возвращает количество гласных букв в строке в любом регистре.

Функция принимает строку, а возвращает число. Гласные — это _aeyuio_.

## Пример

|Ввод|Вывод|
|---|---|
|s = 'How many vOwElS in the phrase?'<br>print(count_vowels(s))|9|

## Примечания

В тестирующую систему сдайте **только код функции** без вызова.

def count_vowels(s):
    a = s.lower()
    number = 0
    for alpha in a:
        if alpha in 'aeyuio':
            number += 1
    return number
