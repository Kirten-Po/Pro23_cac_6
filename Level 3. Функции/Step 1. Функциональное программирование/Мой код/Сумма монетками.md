Напишите функцию, которая собирает требуемую сумму денег монетами разного достоинства, но так, чтобы монеток было как можно меньше.

Функция _coins()_ принимает два аргумента: требуемая сумма и список монет разного достоинства, которые есть в вашем распоряжении. Считается, что монет каждого вида бесконечно много, а сумму точно можно собрать этими монетами.

Функция возвращает список отобранных монет в порядке невозрастания.

### Пример 1

|Ввод|Вывод|
|---|---|
|money = [2, 1, 3]<br>print(coins(13, money))|[3, 3, 3, 3, 1]|

### Пример 2

|Ввод|Вывод|
|---|---|
|money = [1, 7, 5, 4]<br>print(coins(32, money))|[7, 7, 7, 7, 4]|

## Примечания

В тестирующую систему сдайте **только код функции** без вызова.

def coins(n, money):
    lst = []
    for i in range(n):
        b = max(money)
        if n > b:
            n -= b 
            lst.append(b)
        else:
            if n not in money:
                money.remove(b)
            else:
                lst.append(n)
                break
    return lst
