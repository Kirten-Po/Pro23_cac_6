Напишите функцию для выбора наибольших значений списков.

Функция _list_of_largest()_ принимает в качестве аргументов два списка одинаковой длины. Сравнивая элементы списков, стоящие на одинаковых местах, нужно выбирать большее значение. Функция возвращает список из таких больших значений двух списков на каждом месте.

## Пример

|Ввод|Вывод|
|---|---|
|lst1 = [12, 47, 345, 1, 2]<br>lst2 = [13, 6, 346, 42, 0]<br>print(list_of_largest(lst1, lst2))|[13, 47, 346, 42, 2]|

## Примечания

В тестирующую систему сдайте **только код функции** без вызова.

def list_of_largest(lst1, lst2):
    return [max(lst1[i], lst2[i]) for i in range(len(lst1))]
