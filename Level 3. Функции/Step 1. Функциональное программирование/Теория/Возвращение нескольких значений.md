Функция может возвращать несколько значений. Например, из переданного списка нужно выбрать наименьшее и наибольшее значения и вернуть их. Тогда возвращаемые значения просто записываются через запятую (опять же без скобок):

```python
def min_max(lst):
    return min(lst), max(lst)
```

Что в этом случае происходит? Мы уже знакомы с таким типом данных, как кортеж, но привыкли видеть его значения заключёнными в скобки:

```python
tpl = (1, 2, 3, 4)
print(type(tpl))  # <class 'tuple'>
```

Но их можно записать и без скобок, это всё равно будет кортеж:

```python
tpl = 1, 2, 3, 4
print(type(tpl))  # <class 'tuple'>
```

Вот точно так же возвращаемые значения «запаковываются» в кортеж, чтобы мы могли затем присвоить этот кортеж переменной или распаковать при желании:

```python
result = min_max([7, 8, 3, 2, 9])
a, b = min_max([5, 1, 7, 3])
print(result)  # (2, 9)
print(a, b)  # 1 7
```

Будьте внимательны при распаковке: количество переменных слева от знака равенства должно быть таким же, как и число значений справа, чтобы всем хватило значений и ничего лишнего не осталось.

Может ли функция ничего не возвращать? Нет, любая функция что-нибудь возвращает. Даже если мы после return не напишем ничего и даже если вообще не напишем return, возвращаемое значение будет, и это специальный объект — None.

```python
def func():
    return


def func1():
    print('Hello!')


result = func()
print(result)
result1 = func1()
print(result1)
# None
# Hello!
# None
```

При вызове функции func1() сначала выполнилось действие (напечаталась строка), а затем мы вывели возвращаемое значение None.
