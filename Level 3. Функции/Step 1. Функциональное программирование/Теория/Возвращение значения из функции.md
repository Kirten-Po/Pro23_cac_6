Мы написали функцию, которая выводит значение. Но чаще всего в программе значение недостаточно вывести на печать, оно нужно для дальнейших расчётов.

Поэтому придуман механизм _возврата значений_. И мы с ним прекрасно знакомы! Например, встроенная в язык функция len() возвращает длину переданной коллекции. Мы можем присвоить эту длину переменной length:

```python
length = len('Returned value')
```

Говорят, что функция len() вернула число.

Для того чтобы ваша функция вернула значение, нужно перед ним написать служебное слово return. Вот так можно рассчитать сумму переданных в функцию аргументов и вернуть результат:

```python
def adding(a, b):
    c = a + b
    return c
```

Но результат можно считать прямо при возврате значения, то есть обойтись без промежуточной переменной:

```python
def adding(a, b):
    return a + b
```

А вот так мы этот результат получим в программе и присвоим переменной для дальнейшего использования (например, для печати):

```python
result = adding(2, 5)
print(result)  # 7
```

В соответствии с правилами PEP8 после return должен стоять один пробел, а затем возвращаемое значение. И, заметьте, никаких скобок!
