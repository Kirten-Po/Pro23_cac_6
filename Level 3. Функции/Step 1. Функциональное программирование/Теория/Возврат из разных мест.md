В функции может быть не один возврат, а несколько, и это очень удобно: можно выйти из функции, если известен результат, не дожидаясь конца вычислений.

## Пример решения задачи

Рассмотрим задачу, в которой нужно написать функцию **right_sum()**, суммирующую элементы списка, пока не будет получено число, кратное трём, пяти или семи; в этом случае функция возвращает найденную сумму. Но если в процессе суммирования получилось отрицательное число, то нужно вернуть None и прекратить вычисления.

None также возвращается, если подходящую сумму найти не удалось.

При решении задачи видим, что выйти из функции можно в трёх местах:

- если промежуточная сумма отрицательна;
- если нашли подходящую сумму;
- если числа в списке закончились.

Решить можно так:

```python
def right_sum(lst):
   s = 0
   for element in lst:
       s += element
       if s < 0:
           return None
       if s % 3 == 0 or s % 5 == 0 or s % 7 == 0:
           return s
   return None
```
