Функция может возвращать другую функцию в зависимости от переданного аргумента. В этом примере возвращаемые функции находятся «снаружи», в глобальной области видимости.

Например, функция func() принимает строку — символ операции — и возвращает соответствующую функцию, а сами функции определены в программе:

```python
def sum_a_b(a, b):
    return a + b


def mul_a_b(a, b):
    return a * b


def sub_a_b(a, b):
    return a - b


def func(operation):
    if operation == '+':
        return sum_a_b
    if operation == '*':
        return mul_a_b
    return sub_a_b


print(func('+')(5, 3))  # 8
print(func('*')(5, 3))  # 15
print(func('-')(5, 3))  # 2
```

Здесь, чтобы получить ответ, возвращаемое значение функции func снова нужно вызвать, передав два аргумента для выполнения операции.

Также функция может находиться «внутри» функции, принимать её аргументы и в зависимости от них менять своё поведение.

```python
def outer_func(n, *args):
    def inner_func():
        p = n
        for arg in args:
            p *= arg
        return p


    return inner_func


product = outer_func(1, 2, 3)
print(product())  # 6
```
