Для сортировки, да и в других случаях, нам пришлось писать очень короткие функции, буквально в одну строку, а сами такие функции часто использовались только в одном месте. Есть способ проще и короче: надо написать безымянную функцию — lambda-функцию.

Пусть нам нужно отсортировать список строк в лексикографическом порядке, но без учёта регистра. Можно написать функцию для ключа сортировки, которая принимает слово, а возвращает его же в нижнем регистре. Вот так:

```python
def criterion(word):
    return word.lower()


lst = ['Eeny', 'meeny', 'miny', 'Moe',
       'Catch', 'the', 'Tiger', 'by', 'the', 'Toe']
print(sorted(lst, key=criterion))
```

А можно написать лямбда-функцию прямо в ключе:

```python
print(sorted(lst, key=lambda x: x.lower()))
```

Лямбда-функция, как и любая другая функция, может принимать один или несколько аргументов, а может не принимать их вообще. В этом примере аргументом является переменная x, а после двоеточия мы записали, что делать с этим аргументом при возвращении результата.

Есть способ решить эту задачу ещё короче, ведь необходимая нам встроенная функция уже есть, точнее это метод строки, вот его ключу можно и передать:

```python
print(sorted(lst, key=str.lower))
```

Лямбда-функция может не принимать аргументы, тогда после ключевого слова lambda ничего не пишем, сразу формируем возвращаемое значение.

(В этом и следующих примерах мы лямбда-функцию записываем в переменную, что выглядит странно, ведь суть этой функции в том, что она безымянная, а мы даём ей имя... Но для примера это не самый плохой приём.)

```python
b = lambda : True
print(b())
```

Далее показан пример, когда лямбда-функция принимает два аргумента (столько же ей надо передать при вызове):

```python
b = lambda x, y: x * y
print(b(3, 5))
```

Ещё пример.

Помните, в задаче на сортировку строк по количеству гласных мы тоже писали отдельную функцию? Давайте перепишем через лямбду:

```python
lst = ['one', 'two', 'three', 'four', 'five', 'six', 'seven']
print(sorted(lst,
             key=lambda x: len(
                 [y for y in x if y.lower() in 'aeyuio'])))
```

Здесь x в лямбда-функции — это элемент списка, y — буква в этом элементе списка, то есть в строке.
