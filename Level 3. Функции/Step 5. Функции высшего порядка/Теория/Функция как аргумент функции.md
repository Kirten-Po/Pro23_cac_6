Функции, которые принимают другие функции как аргументы или возвращают функции, называются _функциями высшего порядка_.

Рассмотрим, как использовать функцию в качестве аргумента.

## Пример задачи с аргументом-функцией

Напишем функцию **func()**, которая принимает произвольное число аргументов-чисел и именованный аргумент order — функцию для сортировки. Функция возвращает список чисел без повторений в порядке сортировки, определяемый переданной функцией.

Саму функцию func() написать несложно, просто передадим значение аргумента order ключу сортировки.

Интереснее придумать пример, на котором будем проверять работу нашей функции.

```python
def func_to_sort(n):
   return n % 7


def func(*args, order):
   return sorted(set(args), key=order)


arr = [123, 96, 47, 19, 29, 24, 96]
print(func(*arr, order=func_to_sort))
```

В этом примере список, полученный из множества, чтобы убрать повторения, сортируется по остаткам от деления чисел на 7. Результат может оказаться таким:

[29, 24, 123, 96, 47, 19]
