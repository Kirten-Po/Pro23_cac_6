Вы уже написали много программ, вспомните, наверняка когда-то вы допустили такую ошибку — забыли у функции написать скобки, например так:

```python
line = input
print(line)
```

Ваша программа работала, но совсем не так, как вы думали. Вместо того чтобы ожидать пользовательский ввод, программа вывела что-то в угловых скобках:

```python
<built-in function input>
```

Что можно перевести как _«встроенная функция input»_.

Пришло время разобраться, что собой представляет функция в языке _Python_.

В _Python_ всё есть объект: переменная, значение, тип данных, функция. Это значит, что функцию можно записать в переменную — вот точно так, как мы сделали, присвоив переменной line значение input. Теперь у нас две функции, которые могут считывать пользовательский ввод. Попробуем:

```python
line = input
s = line()
print(s)
```

Работает!

Значит, функцию можно передать в другую функцию как аргумент, например, чтобы отсортировать коллекцию или проверять выполнение условий.
