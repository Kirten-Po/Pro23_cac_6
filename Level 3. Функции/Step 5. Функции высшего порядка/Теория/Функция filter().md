Ещё одна встроенная в _Python_ функция высшего порядка — filter() — предназначена для выбора значений из коллекции по условию, определённому функцией-аргументом.

Синтаксис filter():

```python
filter(<функция, выбирающая значения>, <коллекция элементов>)
```

Что значит, что функция выбирает значения из коллекции? Это значит, она должна для каждого элемента вернуть True или False — выбрать этот элемент или не выбирать.

Функция также возвращает итератор.

![](https://yastatic.net/s3/lyceum/files/a0739b17-24dd-43b3-94fe-dd51b9a6dc1f/upload.png)

Например, из списка выберем только слова с нечётной длиной:

```python
lst = ['one', 'two', 'three', 'four', 'five', 'six', 'seven']
print(list(filter(lambda x: len(x) % 2, lst)))
# ['one', 'two', 'three', 'six', 'seven']
```

Из списка чисел выберем только кратные трём:

```python
print(*filter(lambda x: x % 3 == 0, [13, 12, 21, 47, 96]))
# 12 21 96
```

Из букв строки выберем только те, что лексикографически больше буквы n в любом регистре:

```python
print(*filter(lambda x: x.lower() > 'n', 'NeVeRlAnD'))
# V R
```

Часто обе функции применяют одновременно: с помощью filter() отбирают, а с помощью map() преобразовывают.

Прочитаем все числа, записанные в строке через точку, выберем только трёхзначные и преобразуем их в целую часть от деления на 7. Отсортируем в убывающем порядке.

```python
numbers = '21.212.175.79.281.131'
print(sorted(
   map(lambda num: num // 7,
       filter(lambda x: 100 <= x <= 999,
              map(int, numbers.split('.')))),
   reverse=True))
# [40, 30, 25, 18]
```
