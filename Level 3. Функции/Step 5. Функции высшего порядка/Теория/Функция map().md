В Python есть встроенные функции высшего порядка, которые принимают функцию в качестве аргумента. Примером такой встроенной функции является функция для преобразования элементов коллекции в соответствии с переданной функцией-аргументом — map(). Именно эта функция и будет преобразовывать значения.

Синтаксис map():

```python
map(<функция, преобразующая значения>, <коллекция элементов>)
```

Возвращает map() итератор (что это такое, подробнее поговорим чуть-чуть позже), поэтому для того, чтобы увидеть возвращаемые значения, нужно его перебрать поэлементно в цикле, или распаковать, или преобразовать в список.

Пример применения map() для преобразования всех элементов коллекции в их длину:

```python
lst = ['one', 'two', 'three', 'four', 'five', 'six', 'seven']
print(*map(lambda x: len(x), lst))
# 3 3 5 4 4 3 5
```

в верхний регистр:

```python
print(*map(lambda x: x.upper(), lst))
# ONE TWO THREE FOUR FIVE SIX SEVEN
```

в перевёрнутую строку:

```python
print(list(map(lambda x: x[::-1], lst)))
# ['eno', 'owt', 'eerht', 'ruof', 'evif', 'xis', 'neves']
```

Функцию map() можно использовать для считывания чисел, записанных в одну строку:

```python
print(list(map(int, input().split())))  # 172 0 0 1
# [172, 0, 0, 1]
```
