Пушистыми будем считать четырёхзначные числа, у которых в старшем и в младшем разряде стоят единицы, а оставшиеся две цифры образуют простое число.

Выберите из введённой строки чисел, записанных через пробел, пушистые числа и выведите их в порядке возрастания предпоследней цифры через пробел. В случае одинаковых предпоследних цифр порядок не меняется.

Используйте _filter()_.

## Пример

|Ввод|Вывод|
|---|---|
|131 1011 1311 1971 2011 1431 1111|1311 1111 1431 1971|

def simple_digit(x):
    delit = [2, 3, 5]
    d = int(str(x)[1:-1])
    if d != 1 and all(d % g for g in delit):
        return True
    else:
        return False


b = input().split()
digit4 = filter(lambda x: 1000 <= x <= 9999, map(int, b))
digit1_1 = filter(lambda x: x // 1000 == 1 and x % 10 == 1, digit4)
push = filter(simple_digit, digit1_1)
print(*sorted(push, key=lambda x: str(x)[-2]))
