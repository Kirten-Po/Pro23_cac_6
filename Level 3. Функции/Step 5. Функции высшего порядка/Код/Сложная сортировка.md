Напишите функцию, сортирующую произвольное количество переданных ей позиционных аргументов-строк.

Функция _complex_sort()_ принимает произвольное число строк, а возвращает отсортированный список: сортировка сначала по количеству использованных в слове различных букв независимо от регистра, затем по длине слова, затем по алфавиту.

Используйте _sorted()_ с ключом.

## Пример

|Ввод|Вывод|
|---|---|
|print(complex_sort('crisp', 'cool', 'eves', 'with', 'harvest', 'moons'))|['cool', 'eves', 'with', '|

def complex_sort(*a):
    d = sorted(a, key=lambda x: (len(set(x.lower())), len(x), x))
    return d
