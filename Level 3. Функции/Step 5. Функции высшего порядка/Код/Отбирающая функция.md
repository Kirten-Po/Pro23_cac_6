Напишите функцию _selection()_, которая принимает произвольное число позиционных аргументов и именованный аргумент `choice` — функцию, которая возвращает значение `True`, если элемент выбирается, и `False`, если он не подходит. Для проверки элемент нужно передать в функцию `choice` при вызове.

Функция возвращает список отобранных позиционных аргументов в исходном порядке.

def selection(*a, choice):
    lst = []
    for item in a:
        if choice(item):
            lst.append(item)
    return lst
