Напишите функцию _translation()_, которая принимает произвольное количество натуральных чисел и именованный аргумент `base` со значением по умолчанию 2 и возвращает map-итератор, в котором каждое число переведено в 2, 8 или 16-теричную систему счисления в зависимости от значения аргумента `base` (этот аргумент принимает только перечисленные значения).

### Пример 1

|Ввод|Вывод|
|---|---|
|print(*translation(23, 38, 112, 1024, 20, 12, 2011))|10111 100110 1110000 10000000000 10100 1100 11111011011|

### Пример 2

|Ввод|Вывод|
|---|---|
|print(*translation(64, 6487, 253, base=16))|40 1957 fd|


def translation(*a, base=2):
    if base == 2:
        return map(lambda x: bin(x)[2:], a)
    elif base == 16:
        return map(lambda x: hex(x)[2:], a)
    elif base == 8:
        return map(lambda x: oct(x)[2:], a)
