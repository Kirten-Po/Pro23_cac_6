Напишите функцию для игры в города, которая проверяет, начинается ли следующее название города на последнюю букву предыдущего.

Функция _city()_ принимает аргумент — название города. Если это не первый вызов, то функция должна вернуть булево значение (`True` или `False`): начинается ли это название на последнюю букву города, который передавался при прошлом вызове функции. Если же это первый вызов, то нужно вернуть `True`.

## Пример

|Ввод|Вывод|
|---|---|
|print(city('London'))<br>print(city('Westminster'))<br>print(city('Derby'))<br>print(city('York'))<br>print(city('Kingston upon Hull'))<br>print(city('Liverpool'))|True<br>False<br>False<br>True<br>True<br>True|


i = 0
end = ''


def city(n):
    global i, end
    i += 1
    if i == 1:
        end = n[-1]
        return True
    elif n[0].lower() == end:
        end = n[-1]
        return True
    else:
        end = n[-1]
        return False
