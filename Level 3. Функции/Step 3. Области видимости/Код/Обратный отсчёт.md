Напишите функцию _countdown()_, которая ведёт обратный отсчёт старта: при первом вызове возвращает число 10, при каждом следующем — на 1 меньше, пока не дойдёт до нуля, в этом случае нужно вернуть `START`, как и во всех последующих вызовах.

## Пример

|Ввод|Вывод|
|---|---|
|print(countdown())<br>print(countdown())<br>print(countdown())<br>print(countdown())|10<br>9<br>8<br>7|
n = 10


def countdown():
    global n
    if n > 0:
        copy = n
        n -= 1
        return copy
    else:
        return 'START'
