Умная колонка Алиса принимает задания: когда нужно приготовить кофе, когда выключить свет, когда открыть шторы. И записывает задания в глобальный словарь `tasks`.

Напишите функции для постановки новой задачи и удаления задачи при выполнении.

Функция _new_task()_ имеет два аргумента: ключ словаря и время выполнения (в этой задаче — просто целое число). Если такого ключа во внешнем словаре нет, то он создаётся, и время записывается в список значения словаря по этому ключу. Если же ключ есть, то время добавляется в список по этому ключу.

Функция _done()_ также принимает ключ и время. Она должна удалить время из списка по этому ключу, если он есть в словаре. Если в списке ничего не остаётся, то ключ также удаляется.

Обе функции ничего не возвращают, только изменяют словарь.

Для корректной проверки, запишите словарь `tasks` в своём решении до объявления функций с произвольными значениями или пустой.

Не используйте инструкцию _global_ в решении!

|---|---|
|tasks = {}<br><br><br>new_task('make coffee', 5)<br>print(tasks)<br>new_task('turn off the light', 3)<br>print(tasks)<br>new_task('make coffee', 4)<br>print(tasks)<br>done('make coffee', 4)<br>print(tasks)<br>done('turn off the light', 3)<br>print(tasks)|{'make coffee': [5]}<br>{'make coffee': [5], 'turn off the light': [3]}<br>{'make coffee': [5, 4], 'turn off the light': [3]}<br>{'make coffee': [5], 'turn off the light': [3]}<br>{'make coffee': [5]}|


tasks = {}


def new_task(k, v):
    tasks[k] = tasks.get(k, [])
    tasks[k].append(v)


def done(k, v):
    if k in tasks:
        if v in tasks[k]:
            tasks[k].remove(v)
        if len(tasks[k]) == 0:    
            del tasks[k]
