Точно так же, как разделяются области видимости основной программы и функции, можно разделить область видимости основной функции и функции, которая написана внутри основной.

Сейчас поясним.

Напишем функцию, которая принимает аргументы: букву и произвольное число слов. Вернуть функция должна список слов, в которых эта буква встречается не менее двух раз, причём хотя бы один индекс этой буквы чётный. Решение может быть таким: внутри основной функции напишем вспомогательную функцию для определения истинности условия.

```python
def word_filter(letter, *words):
    def condition(s, word):
        count = word.count(s)
        even_indexes = [i for i in range(0, len(word), 2)
                        if word[i] == s]
        return count >= 2 and even_indexes

   result = [x for x in words if condition(letter, x)]
   return result


lines = ['table', 'anagram', 'age', 'acrobat', 'boatman']
print(word_filter('a', *lines))
```

Здесь мы видим целых три области видимости: глобальную (основная программа), локальную для функции word_filter() и локальную для функции condition(). Чтобы получить словари с переменными и их значениями каждой области видимости, можно воспользоваться функцией locals().

Область видимости внешней функции ещё называют нелокальной (nonlocal) по отношению к внутренней функции.
