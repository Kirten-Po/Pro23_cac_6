Давайте теперь рассмотрим, в каких случаях изменить значение глобальной переменной можно и без инструкции global (мы говорили, что её нужно использовать разумно).

Если переменная, значение которой нужно изменить, имеет неизменяемый тип, то без global её изменить нельзя никак. Но если это _изменяемая_ переменная, например список, то можно изменить его содержимое или отдельные элементы.

## Пример поэлементного изменения списка

Пусть требуется написать функцию change_flower(), которая принимает число n и строку line и меняет элемент внешнего (глобального) списка words с индексом n на переданную строку (если такой индекс в списке есть; если нет, то ничего не происходит).

Функция ничего не возвращает, она только меняет переменную words.

Как и при изменении элементов любого списка, к нему нужно обратиться по индексу и присвоить новое значение:

```python
flowers = ['carnation', 'lavender', 'periwinkle',
           'primrose', 'violet', 'fuchsia']


def change_flower(n, line):
    if n < len(flowers):
        flowers[n] = line


change_flower(4, 'daffodil')
print(flowers)
```

Почему же это сработало? Ведь список глобальный! Но мы и не меняли весь список — в этом можно убедиться, если сравнить id списка flowers до и после изменений, — мы изменили только отдельный элемент.

Тогда должны работать и другие поэлементные операции: append(), extend(), pop() и другие. Проверим:

## Пример поэлементных операций

Напишем для примера несколько функций, демонстрирующих некоторые такие операции (но не все, конечно).

```python
flowers = ['carnation', 'lavender', 'periwinkle',
           'primrose', 'violet', 'fuchsia']


def add_flower(line):
    flowers.append(line)


def add_flowers(*lines):
    flowers.extend(lines)


def del_flower(n=-1):
    return flowers.pop(n)


add_flower('thistle')
add_flowers('shamrock', 'camellia', 'aster')
print(del_flower(3))
print(flowers)
```

Ещё один способ изменить содержимое внешнего списка — это заменить его на содержимое другого списка, используя полный срез.

## Пример замены содержимого внешнего списка

```python
flowers = ['carnation', 'lavender', 'periwinkle']


def change(*args):
    flowers[:] = args[:]


change('shamrock', 'camellia', 'aster', 'rose')
print(flowers)
# ['shamrock', 'camellia', 'aster', 'rose']
```
