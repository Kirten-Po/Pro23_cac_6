|   |   |
|---|---|
|Ограничение времени|1 секунда|
|Ограничение памяти|64 Мб|
|Ввод|стандартный ввод или input.txt|
|Вывод|стандартный вывод или weather.json|

Вводятся строки, в которых записаны погодные параметры и их значения через двоеточие и пробел.

Запишите их в виде словаря в файл **weather.json**.

Числовые значения записывайте в виде целых чисел. Форматировать словарь не надо.

## Пример

|Ввод|Вывод|
|---|---|
|temperature: -3<br>precipitation: snow<br>humidity: 80<br>pressure: 740|{"temperature": -3,<br> "precipitation": "snow",<br> "humidity": 80,<br> "pressure": 740}|

## Примечания

В примере вывода показано содержимое файла.

import sys
import json


data = sys.stdin.readlines()
data_pre = [x.rstrip().split(': ') for x in data]
dic = {}
for x in data_pre:
    if x[1].isalpha():
        dic[x[0]] = x[1]
    else:
        dic[x[0]] = int(x[1])
with open('weather.json', 'w') as out_file:
    json.dump(dic, out_file)
