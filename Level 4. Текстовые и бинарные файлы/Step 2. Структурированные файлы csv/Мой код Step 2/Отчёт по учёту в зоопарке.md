В зоопарке провели инвентаризацию и получили данные в виде:  
_<вольер>, <вид животного>, <кличка>, <возраст>_

Ваша задача — прочитать эти данные из потока ввода и записать в файл **report.csv** с заголовками (разделители — точка с запятой):  
_номер по порядку, кличка, вид, возраст, вольер_  
_no, name, mode, age, aviary_

Данные должны быть записаны в порядке: сначала по вольеру, затем по виду, затем по возрасту по убыванию, затем по кличке.

## Примечания

В примере вывода показано содержимое файла.

import csv
import sys


reader = csv.reader(sys.stdin, delimiter=',', skipinitialspace=True)
headers = ['aviary', 'mode', 'name', 'age']
temp = []
for item in reader:
    temp.append({headers[i]: item[i] for i in range(len(headers))})
temp.sort(key=lambda x: (x['aviary'], x['mode'], -int(x['age']), x['name']))
head = ['no', 'name', 'mode', 'age', 'aviary']
for n, row in enumerate(temp, start=1):
    row['no'] = n
with open('report.csv', 'w', newline='') as file_out:
    writer = csv.DictWriter(file_out, delimiter=';', fieldnames=head)
    writer.writeheader()
    writer.writerows(temp)
