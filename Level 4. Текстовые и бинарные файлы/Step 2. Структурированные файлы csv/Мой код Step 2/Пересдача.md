В файле **marks.csv** записана информация о результатах экзаменов студентов в весеннюю сессию. Заголовки файла (разделители — двоеточие):  
_id, фамилия, имя, номер группы, предмет1, предмет2, предмет3, предмет4, предмет5_  
_id, surname, name, group, subject1, subject2, subject3, subject4, subject5_

К пересдаче допускаются студенты, получившие не более двух двоек. Составьте и запишите в файл **retake.csv** ведомость студентов, допущенных к пересдаче. Заголовки (разделители — запятые):  
_id, фамилия, имя, номер группы, предмет для пересдачи_  
_id, surname, name, group, subject_

Если студент пересдаёт два предмета, то для каждого делайте отдельную запись. Записи должны следовать в порядке увеличения id, затем по предмету по алфавиту.

## Примечания

В примере ввода и вывода показано содержимое файлов.


import csv
import sys


with open('marks.csv', 'r', encoding='utf-8') as file_in:
    reader = csv.DictReader(file_in, delimiter=':')
    digit = '2'
    lst = list(reader)
    manage = filter(lambda x: 1 <= list(x.values()).count('2') <= 2, lst)
    result = list(manage)
    keys_need = ['id', 'surname', 'name', 'group']
    data = []
    for item in result:
        for k, v in item.items():
            if k not in keys_need and v == digit:
                line = {field: item[field] for field in keys_need}
                line['subject'] = k
                data.append(line)
    in_field = ['id', 'surname', 'name', 'group', 'subject']
    data.sort(key=lambda x: (x['id'], x['subject']))
with open('retake.csv', 'w', newline='') as file_out:
    writer = csv.DictWriter(file_out, delimiter=',', fieldnames=in_field)
    writer.writeheader()
    writer.writerows(data)
