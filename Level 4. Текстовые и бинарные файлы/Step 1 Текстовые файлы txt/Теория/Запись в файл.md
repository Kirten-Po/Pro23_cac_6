Для записи в файл сначала его нужно открыть в режиме `'w'` или `'wt'`, если это текстовый файл. Если вы этого не сделаете, файл откроется для чтения (по умолчанию), и записать в него ничего будет нельзя.

При записи файла так же, как и при чтении, адрес можно указывать относительно текущей директории, в которой находится файл программы (относительный путь), а также можно указать абсолютный путь к файлу на этом компьютере.

Есть два способа записать информацию в текстовый файл.

**Первый** — использовать метод write() файлового объекта. В этом случае единственный передаваемый в метод аргумент должен быть обязательно строкового типа.

```python
f = open('result.txt', 'w')
s = 'Several strings\nTo write in the\nwriteable file.'
f.write(s)
```

Если такого файла не было по указанному пути, то он будет создан. Если был, то всё содержимое сотрётся и перезапишется новое.

В примере показано, как можно с помощью метода write() записать в файл несколько строк. Формально мы ничего не нарушили, в метод передана одна строка. Разделитель строк — это такой же символ, как и все остальные.

К этому же первому способу отнесём запись списка строк с помощью метода writelines(): он принимает список строк и записывает их в файл; если нужно каждую из них записать с новой строки, то в конец каждой нужно дописать символ перевода строки.

```python
f = open('result.txt', 'w')
s = ['Several strings\n', 'To write in the\n', 'writeable file.\n']
f.writelines(s)
```

**Второй** способ — с помощью обычной функции print(), у которой, оказывается, есть ещё один параметр file, которому и передаётся файловый объект, открытый для записи.

```python
f = open('result.txt', 'w')
s = ['Several strings', 'To write in the', 'writeable file.']
print(s, file=f)
# ['Several strings', 'To write in the', 'writeable file.']
```

Таким образом в файл можно записать всё что угодно, а не только строки. Но всё равно это не объект языка _Python_, а строка. Как сохранять в файлах объекты, вы узнаете немного позже.

Кстати, посмотрите, есть ли в созданных программно файлах последняя пустая строка? Конечно, есть. Файл создаётся правильно.

Чтобы дозаписать информацию в файл, который уже содержит данные, нужно открывать его в режиме `′a′` (дозаписи).

```python
f = open('result.txt', 'a')
s = 'One more string to file.'
print(s, file=f)
# ['Several strings', 'To write in the', 'writeable file.']
# One more string to file.
```

Если файла, открытого в режиме дозаписи, не было, он также будет создан.
