Соберите слово из первых букв всех слов, записанных в файле _lines.txt_. Файл будет доступен вашему решению.

Выведите полученное слово.

## Пример

| Ввод                                                              | Вывод  |
| ----------------------------------------------------------------- | ------ |
| Programming<br>Your<br>Thoughts<br>Help<br>Overcome<br>Negativity | PYTHON |

f = open('lines.txt')
s = [x[0] for x in f.readlines()]
s1 = ''.join(s)
print(s1)
