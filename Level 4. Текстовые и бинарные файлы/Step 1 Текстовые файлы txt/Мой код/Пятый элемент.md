|   |   |
|---|---|
|Ограничение времени|1 секунда|
|Ограничение памяти|64 Мб|
|Ввод|стандартный ввод или fifth.txt|
|Вывод|стандартный вывод или element.txt|

Из всех строк, записанных в файле **fifth.txt** выберите пятый по счёту символ и запишите полученную строку в файл **element.txt**. Если это сделать невозможно, например такого элемента в строке нет, то в выходной файл запишите _Error_.

## Пример

| Ввод                                                          | Вывод  |
| ------------------------------------------------------------- | ------ |
| beaches<br>chiseled<br>capuls<br>dangly<br>analogies<br>able! | hello! |

f = open('fifth.txt')
f1 = open('element.txt', 'w')
c = [x.rstrip() for x in f.readlines()]
Er = False
d = []
for line in c:
    if len(line) < 5:
        Er = True
    else:
        d.append(line[4])
if Er:
    print('Error', file=f1)
else:
    print(''.join(d), file=f1)
