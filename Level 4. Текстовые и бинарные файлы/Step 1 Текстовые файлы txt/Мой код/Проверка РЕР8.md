Напишите функцию **check()**, которая проверяет решение, записанное в файле _solve.py_, на соответствие некоторым правилам PEP8:

- между знаками арифметических операций и операндами должны стоять пробелы (сокращённые операторы не рассматривайте, считайте, что в коде есть только +-*/=);
- перед открывающей или закрывающей скобкой не должно быть пробела, а после закрывающей — должен быть либо пробел, либо конец строки, либо двоеточие;
- если в начале строки есть пробелы, то их количество должно быть кратно четырём; если в предыдущей строке есть двоеточие, то следующая должна начинаться с четырёх пробелов.

Функция должна вернуть результат: если код соответствует РЕР8, то вернуть 0, если ошибка соответствует первому условию, то вернуть -1, второму — -2, третьему — -3.

Ошибки проверяются именно в описанном порядке.

### Пример 1

|Ввод|Вывод|
|---|---|
|a = int(input())<br>b = int(input())<br>c =a+b<br>print(c)|-1|

### Пример 2

|Ввод|Вывод|
|---|---|
|a = int(input())<br>b = int(input())<br>if a > b:<br>print(a + b)|-3|

## Примечания

В примерах показан код, записанный в файле _solve.py_.

Ваша функция будет вызываться так:

```
print(check())
```

def check():
    with open('solve.py', encoding='utf-8') as solve:
        opera = ['+', '=', '-', '/', '*']
        bracets = ['(', ')']
        end = [' ', ':', ')', ']', '\n']
        need_space = False
        prev_space = 0
        first = False
        second = False
        tri = False
        for line in solve:
            idx = len(line) - 1
            cur_space = len(line) - len(line.lstrip())
            if cur_space % 4 != 0:
                tri = True
            if need_space:  
                if cur_space <= prev_space:
                    tri = True
                need_space = False
            if line.rstrip().endswith(':'):
                need_space = True
                prev_space = cur_space
            for i, item in enumerate(line):
                if item in opera:
                    if i == 0 or i == idx:
                        first = True
                    if line[i - 1] != ' ' or line[i + 1] != ' ':
                        first = True
                elif item in bracets:
                    if i < idx:
                        if line[i - 1] == ' ' or line[i + 1] == ' ':
                            second = True
                        elif item == ')':
                            if line[i + 1] not in end:
                                second = True
                            
    if first:
        return -1
    elif second:
        return -2
    elif tri:
        return -3
    else:
        return 0
