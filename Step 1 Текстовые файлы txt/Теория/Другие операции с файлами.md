С файлами можно производить и другие операции, вот некоторые из них:

```python
f = open('example.txt')
```

|   |   |
|---|---|
|**Операция**|**Результат**|
|```python<br>print(f.seek(5))  # 5<br>```|переместить указатель на n байт  <br><br>```python<br>print(f.read(1))  # 3<br>```|
|```python<br>print(f.tell())  # 6<br>```|узнать позицию указателя|

Также можно узнать информацию о файле:

```python
f = open('example.txt')
```

|   |   |
|---|---|
|**Операция**|**Результат**|
|```python<br>f.mode<br>```|r|
|```python<br>f.name<br>```|example.txt|
|```python<br>f.writable()<br>```|False|
|```python<br>f.readable()<br>```|True|

Подробнее о тех операциях, которые мы не рассмотрели или рассмотрели недостаточно подробно, читайте в [документации](https://docs.python.org/3/library/io.html).
